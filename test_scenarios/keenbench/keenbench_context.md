Keenbench is a desktop app meant to be an agentic worker with a custom workflow to efficiently work with office files. Think of it as Cursor for office files. 

The app is be already open. Don't try to run it. You are running it on Linux and the window name will bee KeenBench.
Assume that KEENBENCH_DATA_DIR is set to a clean temp directory for this run.
Required API keys exist and are loaded for the run.

The icon resembles a toolbelt over a white background, to bring it to foreground (if not already there) you'll need to click this icon from the sidebar. Don't worry about maximizing or resizing the window, it will be in the correct size already. 

When you have add a file and clicked "Add a file", first click "Home", then go to "src" and from there assume that you are in the base path. Navigate upward folder by folder.
